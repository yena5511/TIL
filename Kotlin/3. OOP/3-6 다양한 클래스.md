## 다양한 클래스를 다루는 방법

#### Data Class

계층간의 데이터를 전달하기 위한 DTO

    데이터(필드)
    생성자와 getter
    equals, hashCode
    toString

> data class 
```kotlin
data class PersonDto(
    val name: String,
    val age: Int
)
```

data 키워드를 붙여주면 equals, hashCode, toString을 자동으로 만들어준다

named argument까지 활용하면 builder pattern을 ㅆ는 것 같은 효과도 있다
