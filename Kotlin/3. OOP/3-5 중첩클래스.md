## 중첩 클래스르 다루는 방법

#### 중첩 클래스의 종류

자바의 중첩 클래스

- Static을 사용하는 중첩 클래스:
클래스 안에 static을 붙인 클래스
밖의 클래스 직접 참조 불가

- Static을 사용하지 않는 중첩 클래스
    - 내부 클래스 (Inner Class):
    클래스 안의 클래스, 밖의 클래스 직접 참조 가능
    - 지역 클래스 (Local Class):
    메소드 내부애 클래스를 정의
    - 익명 클래스 (Anonymouse Class):
    일회성 클래스

1. 내부 클래스는 숨고진 외부 클래스 정보르 가지고 있어,
참조를 해지하지 못하는 경우 메모리 누수가 생길 수 있고, 
이를 디버깅 하기 어렵다.
2. 내부 클래스의 직렬화 형태가 명확하게 정의되지 않아 직렬화에 있어 제한이 있다.
