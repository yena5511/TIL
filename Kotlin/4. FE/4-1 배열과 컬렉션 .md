## 코틀린에서 배열과 컬렉션을 다루는 법

#### 배열

> 출력 방법
```kotlin
fun main(){

    val array = arrayOf(100, 200)

    for (i in array.indices) {

        println("${i} ${array[i]}")
    }

}
```

```kotlin
fun main(){

    val array = arrayOf(100, 200)

    for ((idx, value) in array.withIndex()) {
        println("$idx $value")
    }

}
```

array.indices는 0부터 마지막 index까지의 Range이다

withIndex()를 사용하면, 인덱스의 값을 한 번에 가져올 수 있다

값을 쉽게 추가 가능
